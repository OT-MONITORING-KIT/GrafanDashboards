# Node Dashboard
This Dashboards contains node realted metrics visualization.

# Telegraf Input Plugins
Following the inpute plugins which should be enabled in telegraf to get each metrics in node dashbord panel.
- cpu
- disk
- io
- mem
- net
- system
- swap
- netstat
- processes
- kernel
- diskio
- procstat
